# personal-productivity

Personal productivity scripts, utilities, and agent skills.

I plan to share more of my personal productivity scripts in this repo over time as they get cleaned up and organized.

## Repo Index

### `skills/`

- `meeting-notes/`: Agent skill for extracting meeting notes, summaries, transcripts, and action items from Google Calendar, Google Docs, and Notion meeting notes.

### `cloudflare-workers/`

- `redirects.ts`: Cloudflare Worker that redirects requests from mapped domains to their target paths.

### `raycast-script-commands/`

- `new-chrome-window.applescript`: Raycast script command that opens a new Google Chrome window.

### Top-level files

- `README.md`: Repository overview and index.
- `LICENSE`: Project license.
